# Igray - Интернет Магазин Игр (КОММЕРЧЕСКИЙ ПРОЕКТ)

Современный интерфейс игровой платформы с каталогом игр, пользовательскими профилями и системой рекомендаций.

![Скриншот](https://i.postimg.cc/tJt1g8VC/igray.png)

## Live Demo
[Посмотреть Демо](https://igray-mp1j.vercel.app/)

## Возможности
- **Каталог игр** - фильтрация по жанрам, платформам и рейтингу
- **Поиск игр** - интеллектуальный поиск с автодополнением
- **Система рекомендаций** - персонализированные подборки игр
- **Профили пользователей** - библиотека игр и достижения
- **Адаптивный дизайн** - оптимизирован для всех устройств
- **Темная тема** - комфортный просмотр в любое время суток

## Технологии
- **Frontend**: React 18, JavaScript, HTML5
- **Стилизация**: CSS3, Flexbox, CSS Grid
- **State Management**: React Hooks (useState, useEffect, useContext)
- **Маршрутизация**: React Router
- **API**: Rawg Video Games Database API
- **Деплой**: Vercel
